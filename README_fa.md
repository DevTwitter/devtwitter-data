> مجموعه‌ای گردآوری‌شده از پست‌های مرتبط با برنامه‌نویسی در کانال تلگرام DevTwitter

## 📖 معرفی

این مخزن شامل خروجی JSON کانال عمومی تلگرام «DevTwitter | توییت برنامه نویسی» است، یک جامعه برنامه‌نویسی فارسی‌زبان. این مجموعه داده شامل پست‌ها، لینک‌ها و واکنش‌ها از زمان ایجاد کانال در می 2020 می‌باشد.

## 🚀 ویژگی‌ها

- تاریخچه کامل پیام‌ها از آغاز کانال تا انتهای 2025
- محتوای غنی شامل پست‌های متنی، لینک‌های مقالات برنامه‌نویسی و تصاویر
- داده‌های واکنش نشان‌دهنده مشارکت جامعه
- زمان‌بندی پیام‌ها و اطلاعات ویرایش
- مناسب برای تحلیل داده، تحقیقات NLP یا استخراج محتوا

## 📊 ساختار مجموعه داده

مجموعه داده به فایل‌های JSON جداگانه بر اساس سال (2020 تا 2025) تقسیم شده است. هر فایل دارای ساختار زیر است:

```json
{
  "name": "DevTwitter | توییت برنامه نویسی",
  "media_domain": "https://media.dvtw.ir/",
  "messages": [
    // آرایه‌ای از اشیاء پیام
  ]
}
```

- `name`: نام کانال
- `media_domain`: آدرس پایه برای فایل‌های رسانه در پیام‌ها
- `messages`: آرایه‌ای از اشیاء پیام شامل متن، لینک‌ها، واکنش‌ها و فراداده

## 🔍 کاربردها

- **تحقیقات NLP**: آموزش مدل‌ها بر واژگان برنامه‌نویسی فارسی
- **تحلیل محتوا**: بررسی روندهای برنامه‌نویسی در جامعه توسعه‌دهندگان فارسی‌زبان
- **منابع آموزشی**: ایجاد مجموعه‌هایی از منابع ارزشمند برنامه‌نویسی
- **سیستم‌های پیشنهاددهنده**: ساخت ابزارهایی برای پیشنهاد مقالات برنامه‌نویسی بر اساس محبوبیت

## 📈 آمار

- **تعداد کل پیام‌ها**: بیش از ۸ هزار
- **بازه زمانی**: 2020- 2025
- **دسته‌بندی‌های اصلی**: توسعه وب، DevOps، زبان‌های برنامه‌نویسی، مهندسی نرم‌افزار
- **زبان‌ها**: فارسی، با محتوای فنی انگلیسی
